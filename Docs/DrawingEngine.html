Class DrawingEngine
Provides a drawing engine for rendering shapes and lines onto a canvas displayed in a PictureBox.
Syntax
C#
    public class DrawingEngine : IDisposable
Remarks
The DrawingEngine class allows users to draw various shapes, such as lines, circles, and rectangles, on a virtual canvas. The canvas is displayed in the provided PictureBox control. The engine maintains the current pen position and drawing state, and supports operations such as clearing the canvas and changing the pen color. This class is not thread-safe and should be used on the UI thread.
